# Hotel Booking Cancellation Prediction

This project explores how machine learning methods can be used to predict whether a hotel reservation will be cancelled.

The work was done as part of a study project focused on applying data analysis and machine learning techniques to real-world data. The dataset contains information about hotel reservations, such as booking details, customer characteristics and stay information.

The main goal of the project was to analyse the data, prepare it for modelling and compare several machine learning algorithms in order to determine which model performs best in predicting booking cancellations.

During the analysis, the data was cleaned and prepared through several preprocessing steps such as handling missing values, encoding categorical variables and creating additional features.

Several models were trained and compared, including Logistic Regression, AdaBoost, Gradient Boosting and XGBoost. Among the tested models, XGBoost achieved the best performance, reaching an accuracy of approximately 84.8%.

The repository contains the full notebook with the analysis and implementation, as well as the written report describing the methodology and results.

## Files

`hotel_booking_cancellation_prediction.ipynb` – notebook containing the full data analysis and model training

`hotel_booking_ml_report.docx` – written report explaining the methods and results
